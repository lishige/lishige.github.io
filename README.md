  - 读书百遍者：这个昵称经历了三大阶段：第一阶段为书海遍读客、第二阶段为书读百遍者、第三阶段为读书百遍者，来源于人有从学者，遇不肯教，而云：必当先读百遍。言：读书百遍而义自见。从学者云：苦渴无日。遇言：当以三余。或问三余之意，遇言：冬者岁之余，夜者日之余，阴雨者时之余也。由是诸生少从遇学，无传其朱墨者；
  - 数据星辰：这个目标，来源于为政之德，譬如北辰，居其所而众星共之。
